1. Ley de Enfriamiento de Newton

Fórmula: $$T = T_s + (T_0 - T_s) \cdot e^{-k \cdot t}$$

¿Cómo funciona este simulador?

Este sistema calcula la temperatura final ($T$) de un cuerpo expuesto a un entorno con temperatura constante ($T_s$) tras pasar un tiempo ($t$) en horas[cite: 12]. Utiliza la función exponencial nativa de JavaScript.

Código JS utilizado:

const tempFin = ts + (t0 - ts) * Math.exp(-k * t);
const res = Math.round(tempFin);

2. Calculador de Combinaciones (Sorteo)

Fórmula: $$C(n,r) = \frac{n!}{r! \cdot (n-r)!}$$

¿Cómo funciona este simulador?

Calcula los subconjuntos posibles combinando elementos sin importar su orden[cite: 19]. Evalúa dos grupos por separado de forma iterativa y multiplica ambos resultados para obtener el total general requerido para el sorteo[cite: 20, 26].

Algoritmo de Factorial propio:

function fact(num) {
    if (num === 0 || num === 1) return 1;
    let r = 1;
    for (let i = 2; i <= num; i++) r *= i;
    return r;
}
